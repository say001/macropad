# materials
- 1x Seeed XIAO RP2040
- 8x Through-hole 1N4148 Diodes
- 1x EC11 Rotary encoders
- 8x Blank DSA keycaps
- 8x Cherry MX Switches
- 6x SK6812 MINI-E LEDs
